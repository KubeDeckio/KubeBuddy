What It Checks

  • Node and Pod Health: Failed nodes, pending pods, crash loops

  • AKS Best Practices: Microsoft-guided checks with a single flag
  • Events: Warnings, crash loops, controller errors
  • Networking + Storage: PVCs, services, policies, and reachability
  • Docker Option: Use without PowerShell by running it in a container
